# RMIT Website Data Scraper
## Running on Windows OS
### Install dependancies
```
pip install requests
pip install beautifulsoup4
pip install lxml
```
### Run scraper
```
python main.py
```
## Running on Mac OS
### Install dependancies
```
pip3 install requests
pip3 install beautifulsoup4
pip3 install lxml
```
### Running scraper
```
python3 main.py
```
